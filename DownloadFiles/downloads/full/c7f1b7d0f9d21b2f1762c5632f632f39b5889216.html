WHOIS servers list for all domain types

In the following text file, you can find the list of all WHOIS servers (port 43) which provide information about registered domains. This list contains the WHOIS servers for generic domains (For example: .com, .org, .net, and so on) as well as for country code domains (For example: .sk, .pl, .it, .de, and so on...)
This list is also internally used in the WhoisThisDomain and WhoisCL utilities.

The structure of the file is very simple. Each line contains the domain extension, a single space character, and then the whois server address. For example: 

it whois.nic.it  
je whois.je

In the above example, you can see that in order to get information about domains of italy (for example: xyz.it) , you should connect the whois.nic.it WHOIS server or if you want to get information about .je domain, use the whois.je server.

This servers list if freely available for any use and without any restriction. However, you should avoid from running an automatic script that downloads the servers list very frequently. If you do so, your IP address might be blocked.

If you find a mistake in the list or you know about any WHOIS server that is not listed here, please report to nirsofer@yahoo.com, and the servers list will be updated accordingly.
