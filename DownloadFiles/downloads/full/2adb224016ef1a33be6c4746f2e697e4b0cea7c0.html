Recover lost/forgotten password from Windows 7/XP/Vista/2003/2008

If you lost or forgot a password, but this password is still stored in your Windows operating system, you can use the following tools to recover the password. Some of these tools can also recover the password from external instance of Windows.

  • Passwords of VPN/Internet connection/Dialup (PPTP, L2TP, and PPPOE protocols): You can use Dialupass utility to recover these passwords. This utility works for current running instance of Windows, as well as for Windows on external drive.

  • Passwords of remote network resource: You can use the Network Password Recovery utility to recover network passwords stored by Windows.

  • Passwords of Windows Live Messenger: You can use MessenPass utility to recover the password of Windows Live Messenger, as well as the passwords of other messenger applications.

  • Passwords of email clients like Windows Live Mail and Outlook: You can use the Mail PassView utility to recover these email passwords stored in your Windows OS.
